# 🌟 Lumina AI - Next Gen Assistant

Lumina AI, Mustafa tarafından geliştirilmiş, çok yönlü bir yapay zeka asistanıdır.

## 🚀 Özellikler
* **Akıllı Sohbet:** Gemini 1.5 Flash altyapısı ile hızlı cevaplar.
* **PDF Oluşturma:** Sohbetleri tek tıkla PDF olarak yeni sekmede açar.
* **Multimedya:** Kamera erişimi ve dosya yükleme desteği.
* **Minimalist Tasarım:** Göz yormayan, şık ve modern arayüz.

## 🛠️ Teknolojiler
* Python (Flask)
* JavaScript
* Vercel Deployment
